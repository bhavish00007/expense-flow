# 💰 ExpenseFlow - Full Stack Application

A modern, beautiful, and feature-rich expense tracking application built with the MERN stack (MongoDB, Express, React, Node.js).

## ✨ Features

- 🔐 **User Authentication** - Secure signup and login with JWT
- 📊 **Interactive Dashboard** - Beautiful charts and statistics
- 💵 **Income Management** - Track all your income sources
- 💸 **Expense Tracking** - Monitor your spending by category
- 📈 **Analytics** - Monthly trends and category breakdowns
- 🎨 **Modern UI** - Stunning interface with Tailwind CSS
- 📱 **Responsive Design** - Works perfectly on all devices
- ⚡ **Real-time Updates** - Instant data synchronization.
